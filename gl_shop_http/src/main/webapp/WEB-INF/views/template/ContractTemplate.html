第五条　迟延交货

    如果卖方因自身原因未能按合同规定按时交付所有或部分货物（包括达成一致的文件），则应向买方进行协商处理，并联系长电电商客服为其进行合同时间延长。

第六条　取消合同

（1）买卖双方协商一致进行合同取消，取消合同不扣交易保证金

（2）如果另一方完全因其责任在合同规定的时间期限内未履行其义务，主动提出取消合同，另外一方不同意取消合同，双方协商不一致，可提请平台进行调解，调解不成的按照《长江电商保证金协议》扣除违约方交易保证金并将其支付给另一方。

（3）合同一方单方面取消合同段的，按照《长江电商保证金协议》扣除违约方交易保证金并将其支付给另一方。

第七条　产品质量责任

1、卖方销售的商品质量必须是符合质量标准（国家标准）的合法商品（产品）。

2、卖方承诺承担由于其产品质量出现问题时给买方所造成的损失。

3、平台为买家提供商品验收机会，卸货前买方可抽检商品质量，卸货完毕后买方可全量验收，买家若在验收过程中有任何问题，可以发起议价与卖家进行协商。

4、双方对商品质量有争议的可提交当地检验机构检验。

第八条　赔偿金

1、如果卖方不能交货或不能按合同规定的条件交货，按照《长江电商保证金协议》扣除方交易保证金并将其支付给买方。

2、如果买方不能按合同规定的条件收货，买方应负责向卖方赔偿，按照《长江电商保证金协议》扣除方交易保证金并将其支付给卖方。

第九条　争议的处理

1、本合同受中华人民共和国法律管辖并按其进行解释。

2、本合同在履行过程中发生的争议，由双方当事人协商解决，也可由有关部门调解；协商或调解不成的，按下列第3种方式解决：

（1）提交仲裁委员会仲裁；

（2）依法向靖江市人民法院起诉；

（3）联系长江电商平台客服进行调解处理。

第十条　合同的效力

1、本合同自双方或双方法定代表人或其授权代表人登录长江电商平台，进行合同确认即刻生效。

2、经平台撮合成交的合同双方必须24小时内确认，才能生效。任一方超过24小时没确认的，合同不成立。
